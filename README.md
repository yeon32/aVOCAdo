## 프로젝트 소개

aVOCAdo는 초·중·고등학생들을 위한 영어단어 암기장인 학습 플랫폼 앱 서비스입니다




## 사용한 기술
- iOS Platform (macOS- Xcode), Swift, UIKit
- Firebase (storage, authentication)
- COCOAPODS
